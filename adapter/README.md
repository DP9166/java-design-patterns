# 适配器模式

> 将一个类的接口转换成客户希望的另外一个接口，使得原本由于接口不兼容而不能一起工作的那些类能一起工作。

首先大家要知道一个可能一直被大家忽略的一个问题就是，适配器模式一共有两种，一种是类适配器，另一种叫做对象适配器。这两个是什么东西呢，经过鄙人的一词一句话概括如下：

**类适配器：通过类的继承或者接口的实现来达到适配目的；**

**对象适配器：通过对象的组合来达到适配目的；**
